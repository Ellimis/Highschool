#include <stdio.h>

int main (){
	int a, i, j;
	
	scanf("%d", &a);
    for(i=1; i<=a; i++)
    {
    	for(j=1; j<=a; j++){
    	    if(j<i)
    	    printf(" ");
    	    else
    	    printf("*");
    }
    	printf("\n");

}
}

길이 n이 입력되면 다음과 같은 역삼각형을 출력한다.


예)

n이 5이면
*****
 ****
  ***
   **
    *





입력



길이 n이 입력된다.




출력



역삼각형을 출력한다.




입력 예시 예시 복사


3


출력 예시


***
 **
  *
